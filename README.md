genome2tx
========
Convert positions from genome coordinates to transcript coordinates

## Requirements
<!-- TODO: to requirements.txt -->

- docopt
- pySam
- GTFParse
- yaml

## Installation

```bash
$ git clone https://github.com/hmdlab/genome2tx.git
$ cd genome2tx
$ python setup.py
```

## Usage

```bash
$ genome2tx <gtf> <bam> 
```
